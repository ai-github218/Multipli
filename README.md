# 🧮 掛け算ゲーム (Multiplication Challenge)

ブラウザ上で動作する**即時回答型掛け算トレーニングゲーム**です。  
制限時間内にどれだけ多くの問題を正確に解答できるか挑戦します。

---

## 🎮 ゲーム概要

- 制限時間：30秒  
- 回答は **Enterキーのみ** で操作  
- 回答スピードによってスコア加算  
- レベルアップに応じて問題の桁数が増加  
- 連続正解数（コンボ）と最大連続記録を表示  

---

## 📐 ゲームルール

1. 初期レベルは **1桁 × 1桁**  
2. 正解するとスコア加算  
3. 回答速度に応じたボーナス点  
4. 5問正解ごとにレベルアップ（桁数が増える）  
5. ミスで連続正解数はリセットされる  

### レベル別出題形式

| レベル | 出題形式 |
|--------|----------|
| Lv1    | 1桁 × 1桁 |
| Lv2    | 2桁 × 1桁 |
| Lv3    | 3桁 × 1桁 |
| Lv4    | 4桁 × 1桁 |
| Lv5〜  | レベルごとに桁数増加 |

---

## 💯 スコア計算

回答速度に応じて得点が変動：

- 2秒未満 → **3点**  
- 2〜4秒 → **2点**  
- 4秒以上 → **1点**  

さらに **連続正解数（コンボ）** と **最大コンボ** を可視化。

---

## 🛠 技術スタック

- **HTML**：構造  
- **CSS**：基本スタイル  
- **JavaScript (Vanilla)**：ロジック実装  
  - DOM操作（`getElementById`, `textContent`, `value`）  
  - タイマー処理（`setInterval`）  
  - ランダム数生成（`Math.random`, `Math.floor`）  
  - レベル・スコア・コンボ管理  

---

## ⚙️ 実装ポイント

- **レベル管理**：レベル = 桁数 → `Math.pow(10, level-1)` を利用  
- **スピードボーナス**：問題開始時間からの経過秒を計算しスコアを変動  
- **連続正解管理**：`combo` 変数で追跡、ミスでリセット  
- **最大コンボ記録**：`maxCombo` 変数で保持  
- **Enterキー入力**のみで回答可能にする `keydown` イベントハンドラ
